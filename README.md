# Minimalist NFT Minting DApp

A professional-grade frontend template for NFT collections. This repository provides a clean, "flat" structure for connecting a web application to the Ethereum blockchain to mint digital assets.

## Features
* **Ethers.js v6:** Uses the latest industry-standard library for blockchain interaction.
* **Wallet Connection:** Robust logic for connecting MetaMask and handling account changes.
* **Contract Interfacing:** Easy-to-use hooks for calling `mint` functions and reading contract state.

## Setup
1. **Install Dependencies:** `npm install`
2. **Configure:** Update the `CONTRACT_ADDRESS` and `ABI` in `App.js`.
3. **Run:** `npm start`

## Design Philosophy
This repo follows a strict "flat" file structure, ensuring that all logic, styling, and assets are immediately accessible in the root directory for rapid prototyping.
